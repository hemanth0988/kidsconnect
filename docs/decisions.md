## Phase 1 Tech Baseline
- Web prototype (mobile-only)
- Plain HTML / CSS / Vanilla JS
- Static JSON data (no backend)
- No framework
- No build step
- Hosted via GitHub Pages

## Layout Constraints
- Phone portrait only (no landscape)
- App runs inside a fixed max-width mobile container (375–430px)
- Outside container: purple/blue gradient + iOS-style blur
- No desktop layouts or responsive expansion
- Overlays are modal sheets/overlays, not full pages
